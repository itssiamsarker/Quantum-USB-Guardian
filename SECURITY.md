# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| 1.0.x | ✅ Yes |

---

## Reporting a Vulnerability

If you discover a security vulnerability in **Quantum USB Guardian**, please do **not** disclose it publicly.

Instead:

- Open a private GitHub issue (if applicable).
- Describe the vulnerability.
- Include reproduction steps.
- Include screenshots or logs if available.

We appreciate responsible disclosure and will investigate reported issues as soon as possible.

Thank you for helping improve the security of this project.
